## 💸 환율 조회 앱

: 화면에서 환율 관련 기본 정보를 확인하고, 원하는 나라의 환율을 조회할 수 있는 앱

---

### 🕹️ 사용한 기술

- Retrofit 사용하여 API에서 JSON 데이터 받아옴
- ViewModel과 LiveData 사용하여 안드로이드 lifeCycle 관리: 화면 전환해도 데이터 삭제 X
- 사용한 API = https://open.er-api.com/v6/latest
